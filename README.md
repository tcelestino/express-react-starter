# express-react-starter #

My default setup for Express and React.js web projects

## Includes ##
- **Back-end**
  - Express 3.4
  - Jade
- **Front-end (Bower)**
  - React 0.10
  - Bootstrap 3
- **Build**
  - Gulp 3.6
    - Wrap
    - Less
    - Concat CSS
    - CSSmin
    - Order
    - Clean
    - jsHint
    - Uglify
    - React-tools
    - Rename
    - Autoprefixer
    - LiveReload

## Install ##
- npm install
- gulp
- npm start
